어떤 프레임워크나 패키지 매니저를 사용하든 상관없습니다. WiseOn Shadcn 컴포넌트 설치는 `shadcn add @wds/<component>` 명령어를 실행하고 `<component>` 부분을 원하는 컴포넌트 이름으로 바꾸는 것만큼 간단합니다.

전체 설치 과정은 `shadcn` CLI 도구가 처리하며, 다른 Shadcn 컴포넌트를 설치하는 방식과 동일하게 작동합니다.

## 예제

<CliCommandCode action="run" command="shadcn@latest add @wds/action-button" />

_작동하지 않는 경우 컴포넌트의 JSON 파일 전체 URL을 대신 사용할 수 있습니다._

<CliCommandCode
  action="run"
  command={`shadcn@latest add ${SERVER_URL}/r/action-button.json`}
/>
