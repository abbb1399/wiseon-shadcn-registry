Shadcn과 마찬가지로, 이것은 컴포넌트 라이브러리가 아닙니다. 대신 여러분만의 컴포넌트 라이브러리를 구축하는 데 사용할 수 있는 컴포넌트 모음입니다. 이 컴포넌트들은 접근성과 커스터마이징을 고려해 설계되었으므로, 프로젝트에 쉽게 통합할 수 있습니다.

이 라이브러리의 각 컴포넌트는 웹 애플리케이션을 개발할 때 대부분의 개발자가 직면하는 특정 문제(비동기 액션, 폼 유효성 검사 등)를 해결하기 위해 만들어졌습니다.

추가되었으면 하는 컴포넌트가 있다면, <a href={GITHUB_REPO_URL}>GitHub 저장소</a>에 이슈를 자유롭게 남겨주세요.
